# TradeX - Real-Time Trading Dashboard

A full-stack trading simulation platform built using:

- Spring Boot (Java 21)
- React + Vite
- MongoDB Atlas
- JWT Authentication
- Role-Based Authorization
- WebSocket (Real-Time Updates)
- Dockerized Deployment

## Features

- User Registration & Login
- JWT-based Security
- Role-based access control
- Real-time portfolio updates via WebSocket
- Market simulation engine
- Positions & Holdings management
- Dockerized full-stack architecture

## Tech Stack

Backend:
- Spring Boot
- MongoDB
- JWT
- STOMP WebSocket

Frontend:
- React
- Axios
- SockJS
- STOMP

DevOps:
- Docker
- Docker Compose
- Nginx
